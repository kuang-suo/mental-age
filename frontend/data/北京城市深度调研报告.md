## 综合评语

> 北京是全国**机会密度最高、天花板最高、竞争也最激烈**的城市。央企总部、头部科技公司、顶尖高校和医院高度聚集，人均GDP和应届生起薪均稳居全国第一。但也因此，房价、落户门槛和生活成本均处在全国最高梯队。如果你追求知识的浓度、权力的浓度、财富的浓度，且愿意承受激烈的竞争和漫长的通勤——北京是绝无仅有的选择。如果你追求轻松、安稳、慢节奏——这座城市可能让你备受煎熬。


## 这 5 类人 ✅ 适合北京

| 类型 | 描述 |
|------|------|
| **高学历技术精英** | 清北/985硕博、AI/芯片/金融从业者，市场定价高，天花板无限。 |
| **体制内与央企目标者** | 绝大多数央企总部在北京，体制内晋升空间全国最大。 |
| **学术与科研追求者** | 国家级实验室、院士密度、科研经费总量全国第一。 |
| **文化创意与传媒从业者** | 全国文化中心，出版/影视/演艺/策展资源无可替代。 |
| **有家庭积累的创业者** | 融资渠道多元，政策支持力度大，但启动成本高于任何城市。 |

## 这 5 类人 ❌ 不适合北京

| 类型 | 描述 |
|------|------|
| **追求安稳慢节奏的人** | 通勤漫长、竞争激烈、节奏快，松弛感稀缺。 |
| **纯靠薪资买房的年轻人** | 房价6万+/㎡，薪资再高也难以单打独斗购房。 |
| **对空气质量极度敏感的人** | 秋冬仍有雾霾过程，PM2.5年均约30+微克/立方米。 |
| **不适应极端干燥寒冷的人** | 冬季干燥刺骨、春季沙尘偶发，南方人需较长适应。 |
| **落户需求迫切的人** | 落户门槛全国最高，积分落户6000人/年、居转户无全国统一政策。 |

> ⚠️ **特别说明**：北京的“城六区”（东城/西城/朝阳/海淀/丰台/石景山）与近郊区（通州/大兴/昌平/顺义/房山）差异极大。房价从西城12.8万/㎡到通州3.76万/㎡，同城价差可达3.4倍。环京通勤族（燕郊/大厂/香河）更是一类特殊存在。本报告以北京市全域为研究对象，重点区分各区域差异。

> 普通本科毕业，进了一家新媒体公司。月薪9K税后约7K，合租昌平月租2,800。每天通勤往返约2小时。最大的感受是：北京很精彩，但精彩大多是别人的。周末去国博/798免费，这是北京对穷人最后的温柔。不知道能坚持多久，但至少来了不后悔。
