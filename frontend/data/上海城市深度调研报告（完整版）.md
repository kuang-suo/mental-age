## 综合评语

> 上海是一座**高机会、高压力、高回报**的城市。配套完善度全国顶尖，机遇密度无可比拟，但居住成本极高、生活节奏极快，对希望安稳慢节奏的人极不友好。如果你渴望挑战、追求国际化视野、愿意为收入天花板拼一把——上海可能是你的应许之地。

| 类型           | 描述                                 |
| ------------ | ---------------------------------- |
| **野心家与事业派**  | 不甘现状，追求职业天花板，愿意加班、竞争、折腾。           |

| **创业者与冒险家**  | 有创业想法或已经创业，需要融资、人才、政策扶持。           |

## 这 5 类人 ❌ 不适合上海

| 类型            | 描述                              |
| ------------- | ------------------------------- |
| **追求安稳慢生活的人** | 喜欢小城市的松弛感，周末爬山种花，不喜欢被效率绑架。      |
| **家庭依赖强的用户**  | 父母年老需照顾、孩子需学区资源、家人无法随迁。         |
