## 综合评语

> 成都，一座让奋斗者和生活家都能找到归宿的城市。一面是13条千亿级产业链、超1.4万家高新企业的科创引擎，一面是人民公园喝茶掏耳、抚琴夜市撸串的烟火人间。它不是全国薪资最高的城市，但套均总价160万、连续7年蝉联“中国最佳引才城市”，让它成为年轻人用更低成本换取体面生活的最佳选择之一。如果你的信条是“生活需要奋斗，但奋斗是为了更好生活”——成都可能就是答案。


## 这 5 类人 ✅ 适合成都

| 类型 | 描述 |
|------|------|
| **在意生活性价比的技术人才** | AI、芯片、新能源从业者，薪资虽不如京沪但有更多可支配收入，吃得香、住得起。 |
| **热爱户外与自然的人** | 开车1小时进山徒步滑雪，周末绿道骑行、公园露营，“雪山下的公园城市”触手可及。 |
| **美食驱动型人格** | 火锅串串钵钵鸡、抚琴夜市2.0版——从5元的小吃摊到黑珍珠，成都就是一座长在舌尖上的城市。 |
| **数字经济/游戏/文娱从业者** | 腾讯天美、鹰角网络以及密集的MCN机构，产业链完整，全国文娱人才高地之一。 |
| **寻找工作与生活平衡的“中间代”** | 不想996到秃头，也不甘心彻底躺平，成都提供了一种体面的“中间态”。 |

## 这 5 类人 ❌ 不适合成都

| 类型 | 描述 |
|------|------|
| **单纯追求薪酬天花板的野心家** | 月薪中位数约7200元，多数岗位薪资与京沪深仍有30-50%差距。 |
| **对快节奏和效率有强迫症的人** | “慢生活”的另一面是部分行业效率偏低，急性子的人可能抓狂。 |
| **讨厌阴天的人** | 成都日照时数全国倒数，冬季多阴天和小雨，“蜀犬吠日”不是传说。 |
| **顶尖金融/咨询业从业者** | 外资投行、顶级咨询公司总部集中于京沪深，成都的高端金融基建仍有差距。 |
| **有严重花粉/潮湿过敏者** | 盆地地形冬季湿度大、梅雨季后紧随花粉季，过敏体质者需要注意。 |

> ⚠️ **特别说明**：成都“5+2”主城区（锦江/青羊/武侯/成华/金牛+高新/天府新区）与二三圈层（双流/郫都/温江/龙泉驿/新都/简阳等）在产业、配套和房价上差异显著。高新区均价约2.5万/㎡，而三圈层的青白江仅约0.8-1.0万/㎡，同城差价可达3倍。

### 人口结构三大特征

| 维度 | 现状 | 影响 |
|------|------|------|
| **中心城市虹吸效应** | “5+2”主城区+双流区人口高度密集，双流区以279万人居全市第一 | 医疗、教育资源集中，但也带来了主城区通勤压力 |
| **中度老龄化** | 每5个户籍人口中就有1个超过60岁 | 养老、医疗公共服务持续承压，银发经济兴起 |
| **年轻人净流入** | 连续7年“中国最佳引才城市”，每年可提供20万左右大学毕业生 | 为老城注入活力，新兴产业后备人才充沛 |
