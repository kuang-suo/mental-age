### 后端修改 (backend/src/utils/scoring.js)

#### 1. 维度得分计算公式
**位置**: calculateDimensionScores() 函数，第101行
```javascript
// 改前：1-5 映射到 0-10
scores[dim] = Math.round((avg - 1) * 2.5 * 10) / 10;

// 改后：1-5 映射到 0-5
scores[dim] = Math.round((avg - 1) * 1.25 * 10) / 10;
```

**计算逻辑**:
- 用户答案范围: 1-5
- 维度平均分: (答案1 + 答案2 + ...) / 题目数
- 最终得分: (平均分 - 1) × 1.25
  - 答案都是1时: (1-1) × 1.25 = 0
  - 答案都是5时: (5-1) × 1.25 = 5

#### 2. 心理年龄计算公式
**位置**: calculateMentalAge() 函数，第115行
```javascript
// 改前：基于 0-10 范围
const mentalAge = Math.round(18 + (avgScore / 10) * 42);

// 改后：基于 0-5 范围
const mentalAge = Math.round(18 + (avgScore / 5) * 42);
```

**映射关系**:
- 维度平均分 = (得分1 + 得分2 + ... + 得分8) / 8
- 范围: 0 → 18岁，5 → 60岁

#### 3. 关键词生成阈值
**位置**: generateKeywords() 函数，第161行
```javascript
// 改前：得分 >= 6
if (score >= 6) {

// 改后：得分 >= 3
if (score >= 3) {
```

**原因**: 在0-5范围内，3分代表中上水平，相当于原来的6分
