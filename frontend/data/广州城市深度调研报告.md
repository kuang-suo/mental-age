## 综合评语

> 广州，一座烟火气与野心并存的千年商都。它既有67家三甲医院和7所双一流高校构筑的华南最强民生底盘，也有AI岗位中位年薪超33万元的新兴产业爆发力；既有早茶一盅两件的慢生活，也有琶洲人工智能与数字经济试验区的彻夜灯火。相比北上深，广州的房价诚意为一线城市中最足——套均总价423万、用更低的居住成本换来几乎同等的城市配套。如果你想要一线城市的机会厚度，又不愿被房贷压弯了腰——广州，是性价比最高的答案。


## 这 5 类人 ✅ 适合广州

| 类型 | 描述 |

| **传统+新兴行业“双栖人才”** | 商贸、制造基础雄厚，AI、新能源、低空经济等新赛道也在快速崛起。 |
| **饮食与文化重度爱好者** | 早茶、烧腊、糖水、夜市——广州人对“吃”的讲究深入骨髓，文化底蕴也厚重。 |
| **跨境与外贸从业者** | 广交会链接全球，跨境电商、大宗贸易、供应链行业机会密集。 |
| **对公共资源有刚需的家庭** | 67家三甲+43所国家级示范性高中，教育和医疗在一线城市中性价比最高。 |

## 这 5 类人 ❌ 不适合广州

| 类型 | 描述 |
|------|------|
| **纯追求薪酬天花板的野心家** | 应届生年薪21万，比北京低了约8万，核心赛道薪资仍有差距。 |
| **不喜湿热气候的人** | 夏季长达约5个月，回南天墙壁淌水、体感黏腻。 |
| **对有组织集中供暖有依赖的人** | 冬季湿冷无暖气，体感温度可比北方-5℃更难熬。 |

> ⚠️ **特别说明**：广州“中心四区”（天河、越秀、海珠、荔湾）与外围区域（黄埔、番禺、白云、南沙、花都、增城、从化）差异显著。天河房价5.9万/㎡，而从化仅约0.93万/㎡。选择合适的板块是定居广州的核心命题。

### 战略性新兴产业——广州的新引擎

“十四五”期间，广州战略性新兴产业增加值超万亿元，占GDP比重超过30%。形成了6个千亿级先进制造业集群、10个千亿级服务业。先进制造业增加值占规上工业的比重提升至60%，落地亿元以上产业项目3,300多个，其中百亿级项目34个。广州实验室产出数十项关键成果，全球首个抗甲型流感PB2新药等8款药物和疫苗获批上市。
