## 综合评语

> 厦门是一座“宜居性”与“性价比”存在撕裂感的城市。空气质量全国顶尖，山海交融的花园环境无可挑剔，慢节奏的生活氛围让很多人来了就不想走。但岛内房价比肩一线、薪资水平仅为二线，高房价与低收入的剪刀差，让年轻人在此定居的梦想变得格外沉重。如果你追求生活品质、热爱海洋与文艺，且已有一定的经济基础——厦门会是你理想的家。如果你一穷二白、指望打工在此买房安家——三思而后行。


## 这 5 类人 ✅ 适合厦门

| 类型 | 描述 |
|------|------|
| **追求生活品质的人** | 热爱大海、蓝天、花园城市，愿意为居住环境付费。 |
| **有经济基础的自由职业者** | 网店店主、远程工作者、内容创作者，收入不依赖本地薪资。 |
| **闽南籍在外打拼的“回流族”** | 有家乡情结、家人已购房或可提供居住支持。 |
| **新能源/电子信息/海洋经济从业者** | 厦门的支柱产业从业者，有较好职业平台。 |
| **退休养老群体** | 气候宜人、医疗配套完善，为退休生活提供优质环境。 |

## 这 5 类人 ❌ 不适合厦门

| 类型 | 描述 |
|------|------|
| **纯靠薪资买房的年轻人** | 岛内房价5万+、月薪8000起步，买房遥遥无期。 |
| **追求职业天花板的野心家** | 产业链不如一线完整，薪资天花板明显低于北上深。 |
| **习惯于一线城市效率的人** | 地铁网络密度不够、商业配套类型不够丰富。 |
| **不喜潮湿闷热的人** | 夏季漫长潮湿，梅雨季节空调常开。 |
| **对学区极度焦虑的家长** | 优质教育资源集中在岛内，竞争激烈，房价更高。 |

> ⚠️ **特别说明**：厦门岛内（思明/湖里）与岛外（集美/海沧/同安/翔安）差异极大。岛内单价5-6万/㎡，岛外1.8-2.8万/㎡，生活成本与体验完全不同。建议结合具体区域做选择。

# Part 4 · 交通基础设施 · 教育 · 医疗

## 交通网络

| 指标 | 数据 |
|------|------|
| **已建成地铁** | 3条线路（1/2/3号线），与厦门站、厦门北站、邮轮中心无缝衔接 |
| **在建线路** | 4号线（岛外环湾快线）、6号线等 |
| **BRT系统** | 全国首条高架BRT，与地铁5座换乘站 |

厦门目前已建成3条地铁线路，与厦门北站、厦门站、邮轮中心形成无缝衔接，并与BRT线网形成5座换乘车站。地铁4号线集美软件园北至翔安机场段全长约45公里，设计最高时速120公里，可实现半小时直通厦门北站与翔安机场。地铁3号线机场段可直达翔安国际机场，从厦门火车站搭乘地铁约1小时可达。

### 综合交通枢纽

厦门拥有厦门站（岛内）、厦门北站（集美）、翔安国际机场（建设中）三大交通枢纽。港口连通性指数位列全球第11位。

## 居住压力 —— 核心矛盾

| 指标 | 数据 |
|------|------|
| **房价收入比** | 约21.1（全国第五）|
| **岛内房价** | 思明区约4.8万/㎡、湖里区约4.45万/㎡ |
| **岛外房价** | 集美2.8万/㎡、海沧2.4万/㎡、翔安2.2万/㎡、同安1.9万/㎡ |
| **套均总价** | 约435.9万元 |

2025年9月，厦门住宅均价为35,737元/㎡，套均总价435.9万元。房价收入比约21.1，位列全国第五，仅次于若干一线和强二线城市。

**岛内外分化明显**：思明区二手房挂牌均价约56,100元/㎡（6月数据），而同安区仅约16,467元/㎡，同城价差近3.5倍。岛内核心区（五缘湾等）豪宅站稳6万+/㎡，岛外远郊去化压力大，同安环东海域有楼盘以1.8万/㎡“割肉”出售。

📉 **购房建议**：
- 刚需首选：海沧、翔安、同安的地铁沿线（2-2.5万/㎡，200万内总价）
- 改善升级：岛内老小区小户型（4-5万/㎡，预算400万+）
- 租房过渡：合租1,500-2,500元/月，整租3,500-6,000元/月

# Part 7 · 落户政策 · 定居行动清单 · 真实生活场景

## 厦门四大落户路径

| 路径 | 条件 | 特点 |
|------|------|------|
| **岛外租房落户** | 在集美/海沧/同安/翔安住房租赁备案满半年+社保缴纳满3个月 | 门槛最低，2025年新政 |
| **人才引进落户** | 本科及以上学历+在厦稳定就业 | 最常用通道，含新引进人才生活补贴 |
| **技能人才落户** | 持有高级工及以上职业资格证书+稳定就业 | 技工院校毕业生友好 |
| **投资纳税落户** | 连续3年纳税累计达到一定额度 | 企业家/投资人通道 |

2025年11月起，厦门全面放宽岛外四区落户条件，新增租房落户；同时全面设立社区公共户口。
